# 법인카드 영수증 자동화 프로그램

법인카드 영수증을 업로드하여 OCR로 데이터를 자동 추출하고, 양식을 작성한 후 PDF로 다운로드할 수 있는 웹 애플리케이션입니다.

## 기능

- ✅ **영수증 이미지 업로드**: 드래그 앤 드롭 또는 파일 선택으로 영수증 이미지 업로드
- ✅ **OCR 자동 인식**: Tesseract.js를 사용하여 영수증에서 사용일자와 금액 자동 추출
- ✅ **양식 작성**: 법인카드 영수증 양식에 데이터 입력
- ✅ **PDF 다운로드**: 작성된 양식과 영수증 이미지를 PDF로 다운로드

## 사용 방법

1. **프로그램 실행**
   - `index.html` 파일을 웹 브라우저에서 열기
   - 또는 라이브 서버 사용 (VSCode에서 Live Server 확장 설치 권장)

2. **영수증 업로드**
   - "종이 영수증 업로드" 영역을 클릭하여 파일 선택
   - 또는 영수증 이미지를 드래그 앤 드롭

3. **자동 입력 확인**
   - OCR이 자동으로 사용일자와 품의금액을 입력합니다
   - 필요시 수동으로 수정 가능

4. **양식 작성**
   - **카드번호**: 0414 (자동 입력됨)
   - **사용일자**: OCR로 자동 입력 (수정 가능)
   - **예산과목**: 드롭다운에서 선택
   - **사용부서**: 차량사업소 (자동 입력됨)
   - **사용자**: 직접 입력
   - **지급내역**: 직접 입력
   - **품의일자**: 직접 입력
   - **품의금액**: OCR로 자동 입력 (수정 가능)

5. **PDF 다운로드**
   - 모든 필수 항목 입력 후 "PDF 다운로드" 버튼 클릭
   - PDF 파일이 자동으로 다운로드됩니다

## 기술 스택

- **HTML5**: 웹 페이지 구조
- **CSS3**: 모던하고 반응형 디자인
- **JavaScript (ES6+)**: 애플리케이션 로직
- **Tesseract.js**: OCR (광학 문자 인식)
- **jsPDF**: PDF 생성
- **html2canvas**: HTML을 이미지로 변환

## 주의사항

- 인터넷 연결이 필요합니다 (CDN을 통해 라이브러리를 로드)
- OCR 정확도는 영수증 이미지 품질에 따라 달라질 수 있습니다
- 자동 추출된 값은 반드시 확인하고 필요시 수정하세요
